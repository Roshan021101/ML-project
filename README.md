AeroSphere is a high-fidelity environmental analytics suite designed to monitor and forecast regional Air Quality Index (AQI) levels across India. Built entirely using custom HTML5, native Markdown, and optimized inline CSS3, the platform bypasses brittle web frameworks and external CDNs to deliver an uncompromised, human-crafted user experience that works flawlessly offline. The system features an automated regional lookup matrix covering all major Indian states and districts from A to Z, instantly estimating baseline pollutant metrics behind the scenes to eliminate manual data entry overhead.At its core, the platform routes data through a self-contained three-tier architecture. It extracts a localized four-dimensional parameter array containing Sulphur Dioxide ($SO_2$), Nitrogen Dioxide ($NO_2$), RSPM, and SPM, feeding it directly into a pre-trained multivariate regression pipeline (model.pkl) to compute the final AQI score. The user interface applies glassmorphic styling and an adaptive color-aura routing engine that dynamically scales the container's neon glowing borders from a safe green to a critical dark red. This provides immediate, human-readable visual confirmation of regional atmospheric safety alongside comprehensive quantitative data logs and tailored health advisories.
